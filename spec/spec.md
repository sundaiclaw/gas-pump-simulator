# Gas Pump Simulator

Gas Pump Simulator is an arcade management game about running a chaotic gas station lane.

## What it does
- spawns customers with fuel orders and mood timers
- lets the player pump fuel and stop at the right amount
- punishes overfills, delays, and wrong choices
- tracks score, streak, queue pressure, and station chaos

## Technical brief
- Frontend: React + Vite single-page app
- Backend: optional Node/Express shell, but gameplay can run client-side
- Deployment target: Cloud Run container serving the built app
